
# Repositório do teste técnico para Delta

  
  

## Para iniciar o projeto:

   

 1. yarn ou npm install
 2. yarn start ou npm start

  
## Dependências

 - React: 17.0.1
 - firebase: "8.2.4",
 - react-router-dom: "5.2.0",


## Características:

> Projeto CRUD básico, utilizando Firebase como banco de dados.
